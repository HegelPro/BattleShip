# BattleShip

## Getting Started

### Install all packages

for web-server:
```
cd server
npm i

```
after that for client:
```
cd client
npm i

```

### Run servers

for web-server:
```
cd server
npm run server

```
after that for client:
```
cd client
ng serve

```
